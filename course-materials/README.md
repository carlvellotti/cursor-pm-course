# Cursor for Product Managers - Course Materials

Welcome to the interactive Cursor course for Product Managers!

## Getting Started

**You're in the right place!** This folder contains everything you need to complete the course.

### What's Included

- **`lesson-modules/`** - Interactive modules taught by Claude (Modules 1-3)
- **`company-context/`** - TaskFlow company reference materials for exercises
- **`.cursor/`** - Slash commands and rules that power the course experience

### How to Start the Course

**1. Make sure you opened this folder in Cursor:**
- You should see `.cursor`, `company-context`, and `lesson-modules` in the File Explorer

**2. Open Composer:**
- Press `Cmd+I` (Mac) or `Ctrl+I` (Windows)

**3. Start the first module:**
```
/start-1-1
```

This kicks off Module 1.1 - Welcome to TaskFlow!

### Slash Commands

Throughout the course, you'll use slash commands like:
- `/start-1-1` - Start Module 1.1
- `/start-2-1` - Start Module 2.1
- `/start-3-1-1` - Start Module 3.1.1
- etc.

These commands are defined in the `.cursor/` folder and guide you through each module interactively.
