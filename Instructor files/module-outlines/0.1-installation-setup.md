# Module 0.1: Installation & Account Setup

## Files Needed to Start
- None (pre-course setup)

## Files Created During Module
- None (installation and configuration only)

## High-Level Overview
- Download Cursor from cursor.com (Mac .dmg or Windows .exe)
- Install Cursor (drag to Applications on Mac, run installer on Windows)
- Launch Cursor for the first time
- Create free account with email/password (no credit card required)
- Explain 14-day Pro trial (automatic for all new users)
- Enable codebase indexing
- Mention student offer (1 year free Pro with .edu email at cursor.com/students)
- Cursor is now ready to use
