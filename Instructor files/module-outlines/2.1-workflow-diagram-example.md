# Module 2.1 Workflow Diagram Example

This is what Cursor will display in chat at the beginning of Module 2.1 to show students the complete workflow.

## Mermaid Diagram Code

```mermaid
graph TD
    A[Start: Module 2.1 PRD Writing] --> B[Phase 1: Choose Foundation]
    B --> B1[Choose PRD Template:<br/>Carl's / Lenny's / Your Own]

    B1 --> C[Phase 2: Build Context]
    C --> C1[@mention taskflow-company-context.md]
    C --> C2[@mention pain-points.md user research]

    C1 & C2 --> D[Phase 3: Sharpen Thinking]
    D --> D1[@mention socratic-questioning.md]
    D1 --> D2[Answer 3-5 strategic questions<br/>about the feature]

    D2 --> E[Phase 4: Generate Options]
    E --> E1[Generate 3 PRD versions]
    E1 --> E2[Version 1: Chat-First]
    E1 --> E3[Version 2: List-First]
    E1 --> E4[Version 3: Balanced]

    E2 & E3 & E4 --> E5[Choose preferred version]

    E5 --> F[Phase 5: Multi-Perspective Review]
    F --> F1[@mention reviewers/engineer.md]
    F --> F2[@mention reviewers/executive.md]
    F --> F3[@mention reviewers/user-researcher.md]

    F1 & F2 & F3 --> F4[Consolidated Review Document]

    F4 --> G[Phase 6: Address Feedback]
    G --> G1[Multiple-choice questions<br/>for each feedback item]
    G1 --> G2[Choose approach A, B, or C]
    G2 --> G3[Cursor edits PRD with changes]

    G3 --> H[âœ… Production-Ready PRD]
```

## What Students Will See

When the module starts, Cursor will render this flowchart directly in the chat, giving students a visual map of:
- Where they are in the process
- What comes next
- How all the pieces fit together

This helps answer the critical "Where am I?" question throughout the module.
